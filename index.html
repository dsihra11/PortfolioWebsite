NCDSB Job Posting Website

An application for users to be able to search, view, and apply to job postings created by the administrator. As an admin, you are able to create, edit, and view postings, positions, and applications. They will be able to create postings by batch processing, by template, or from scratch. Template will be created through positions tab. The user has access to their own personal profile, as well as the list of postings available. They are able to upload their resume as well as add covering comments and skills/qualifications. The main postings page is able to be filtered by keywords, as well as the FTE.

My Role

I was one of four contributors to this project. I was the front-end developer, responible for desiging and developing the layout of the page, working with back-end developers and web designers to improve usability, and troubleshoot code.
